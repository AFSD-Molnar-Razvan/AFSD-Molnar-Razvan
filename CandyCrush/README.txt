# Proiect: Automatizare Candy Crush

## Descriere
Acest proiect simulează automat jocul Candy Crush pe o tablă de 11x11. Programul joacă singur 100 de jocuri, încercând să obțină un scor de 10.000 de puncte cât mai rapid.

## Cum se rulează
Nu trebuie instalat nimic. Se folosește doar Python standard.

**Comanda de rulare:**
```bash
python candy_crush.py
